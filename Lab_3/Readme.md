# Lab 3 – Linear Regression and Effect of Standardization

## Objective
Understand how feature scaling affects Linear Regression models.

## Tasks
1. Train Linear Regression without scaling.  
2. Standardize features using StandardScaler.  
3. Retrain and compare model coefficients and performance.  
4. Plot predictions vs. actual values for both models.  

## Key Concepts
- Linear Regression  
- Mean Squared Error (MSE)  
- R² Score  
- Feature Standardization  

## Tools Used
- Scikit-learn (LinearRegression, StandardScaler)  
- Matplotlib  
- Pandas  

## Output
- Regression coefficients (before/after scaling)  
- Performance comparison plots
